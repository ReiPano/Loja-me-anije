# Spaceship Game
A simple and interesting game written in Java.
Media folder must be in the Desktop directory in order for the game to work.
You must chose a difficulty level or the game won't start. You can chose the difficulty at the "Zgjidh" button.
Currently only plays on Windows. Windows 8-10 is tested. Other versions could work as well.
